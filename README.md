### Fintech-company
___
Project for processing loan applications. Was carried out as part of the Tinkoff Backend Academy.

Includes modules:

* Api
* Origination
* Scoring
* Payment-Engine

## Stack

* Spring
* PostgreSQL
* Hibernate
* REST
* gRPC
* Docker
* Gradle
* MapStruct
* Liquibase
* Testcontainers
